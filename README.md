# PROJECT TITLE
A kneaded, B sprinkled, C overbaked

## Short Description
Someone secretly left you a little bread, probably hoping you’ll pass one on to the next visitor. Every loaf here gets a bit of teamwork, someone kneads, someone sprinkles, someone overbakes, yet they always turn out strangely adorable.

## How to Run / Install
1. Install Node.js.
2. Download or clone the project files.
3. Open a terminal in the project folder.
4. Install dependencies: npm install express socket.io
5. Run the server: node server.js
6. Open your browser and go to: http://localhost:3000

Interaction:
1. Create:
Press A to create a new mochi

2. Kneading:
Hold down the mouse button on the mochi
Once the progress bar reaches 100%, the proving stage begins

3. Proving:
Wait for approximately 20 seconds
The mochi will gradually grow in size

4. Decorating:
Once the mochi has finished proving, click on its surface to add sugar sprinkles

5. Baking:
Double-click the mochi to start baking
A progress bar will appear during baking; you can choose to stop once it reaches your preferred level of doneness

6. Removing from the oven / Eating
Once baking is complete, click the mochi
You can eat it, or leave it for the next visitor
